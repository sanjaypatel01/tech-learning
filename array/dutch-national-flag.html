Ducth National Flag Problem
Problem - Sort an array which contains 0, 1 and 2.
Solution :
Approaches --
  1. Sort using merge Sort 
    2.  
    Time complexity - O(n log n)
    Space Complexity - O (n)
  2. Use 3 variables to have count of 0s, 1s and 2s. Later update the array as per count.
    3.     Time Complexity - O(2 n) ~~ O(n) - Works in two loops
    Space Complexity - O(1) - Constant space or no extra space
  3. Optimal Approach - Ducth National Flag Algorithm 
    4.     Time complexity - O(n)  - Works exactly in one loop
    Space Complexity - O(1) - Constant space or no extra space

    Consider having 3 pointers
    low, mid and high
    elements from 0 to low - 1 indices will be having 0.
    elements from low to mid - 1 indices will be having 1.
    elements from mid to high - 1 indices, consider unknown - This will be at the beginning. So basically whole array, as mid will be 0, and high will be n-1
    elements from high to n - 1 indices will be having 2.

    Now iterate array using mid which is 0 initially iterate until it is lesser to high
    Now using switch
    case 0: swap (Array[mid], Array[low]); low++, mid++; // Here low will be at 1 or low and mid both at same index.
    case 1: mid++;
    case 2: swap(Array[mid], Array[high]); high--; // No need to do mid++, as the element as position high, not sure what value it is containing 0, 1 or 2.
    
      import java.util.* ;
      import java.io.*; 
      public class Solution {
        public static void sortArray(ArrayList arr, int n) {
          
          int low = 0;
          int mid = 0;
          int high = n-1;

          while(mid <= high) { // Here operator must be <=
              switch(arr.get(mid)) {
                  case 0:
                      swap(arr, low, mid);
                      low++;
                      mid++;
                      break;
                  case 1:
                      mid++;
                      break;
                  case 2:
                      swap(arr, mid, high);
                      high--;
                      break;
              }
          }
        }
        private static void swap(ArrayList arr, int index1, int index2) {
          int temp = arr.get(index1);
          arr.set(index1, arr.get(index2));
          arr.set(index2, temp);
        } 
      }
