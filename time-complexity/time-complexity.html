Time complexity-

Rate at which time taken increases with respect to the input size.
Time complexity != time taken

Big Oh Notation - O(n) - Worst case - Upper bonud

Rules

  • Worst case scenario
  • Avoid constants
  • Avoid lower values

Theta Notation - θ(n) - Average case

Omega Notation - Ω(n) - Best Case: Lower bonud

Space Complexity - Memory Space

Big Oh Notation - O(n) => Auxilary space + Input space
Auxilary space - The space that you take to solve the problem.
Input space - The space that you take to store the input.
function(int []input) { int c = 12; ... } Here input array is a input Space c is Auxilary space Never ever change input data inside the function, untill you are asked to do so.
